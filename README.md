# Trending News Webpage

## Description
This project is a simple, static webpage that displays trending news articles. It is built using only HTML and CSS, showcasing responsive design principles and modern web layout techniques.

## Table of Contents
- [Usage](#usage)
- [Features](#features)
- [Contributing](#contributing)

### Usage
This webpage can be used to display trending news articles. You can update the content by editing the HTML file, replacing placeholder text with actual news articles and headlines. The design is fully responsive, ensuring it looks great on both desktop and mobile devices.

### Features
- Responsive design using pure CSS
- Simple and clean layout
- Easily customizable HTML structure for updating news content
